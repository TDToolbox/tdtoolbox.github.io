BTD6 Mod Support Added to TD Loader

August 31st, 2020

Since BTD6 mod api was just released, we decided to add support for BTD6 to TD Loader. You can now manage all of your BTD6 mods just like you could before with the other games.

In addition to this, we've redesigned the UI so it looks cleaner, and we've fixed a lot of the bugs that were reported in the previous version. It should be much more stable now!

BTD6 Mod Api Has Been Released!

August 30th, 2020

A mod making api has been released for BTD6! It is now much easier to make mods for BTD6. Thanks to BowDown097 and Danny Parker for their unwavering commitment to bring this to our community.

NKHook5 has been updated to work with latest BTD5

August 27th, 2020

BTD5 recieved a small update on August 20th, bringing with it a new map called Parking Lot. While NKHook5 is made to auto-update every time BTD5 updates, there were a few things that didn't update correctly. Mallis, the creator of NKHook5, fixed those issues and released this latest version.

There are several exciting features being developed, stay tuned for more info.
